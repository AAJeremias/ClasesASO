# Órdenes para la gestión de impresoras y trabajos
## Imprimir un archivo de texto
```Bash
cat imprimir | lpr
```
## Ver cola de impresión
```Bash
lpq
```
## Borrar cola de impresión
```Bash
lprm -
```
